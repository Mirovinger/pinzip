1 - Инструкция
================

1. Загрузите файл inzip.php в корень файловой системы сайта
2. Смените пароль в первой строке файла inzip.php, по умолчанию пароль равен 1234567
3. Откройте в брайзере /inzip.php
4. Выберете нужные файлы и каталоги для архивирования
5. В случае успеха файл будет сохранен рядом с inzip.php с названием backup_дата.zip

В случае если сервер выдал ошибку, попробуйте сократить количество архивируемых данных, либо увеличить время работы сервера в php.ini
